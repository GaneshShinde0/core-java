# The Parts of a Regular Expression

Following table displays some common metacharacters that fall into these categories.

| Type                          | Example|
|-------------------------------|-------------------------------|
 | Character Class               | .[abc][a-g][A-Z][0-9][^abc]\d\s\w|
 | Quantifiers                   | * + ?|
 | Boundary matchers(or anchors) | ^ $ \b|
 | Groups                        | ()|
